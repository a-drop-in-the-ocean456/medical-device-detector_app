## 交互说明

1. **点击上传区域**：弹出选择菜单（拍照/相册）
2. **点击图片**：预览大图
3. **点击重新选择**：清空当前图片和结果
4. **点击更换图片**：重新选择图片来源
5. **点击复制**：复制检测结果到剪贴板
6. **点击服务器设置**：展开/收起设置面板
7. **点击测试连接**：验证服务器连接状态

## 响应式设计

- 适配各种屏幕尺寸
- 图片自适应宽度
- 按钮在窄屏下自动调整
- 文字大小使用rpx单位
